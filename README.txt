Древо рода Маргошвили
=====================

Есть два режима работы.

1. Обычный оффлайн-режим
------------------------
Откройте index.html двойным щелчком.

В этом режиме сайт работает без интернета и без backend, но данные сохраняются
только в браузере этого компьютера. Другие компьютеры эти изменения не увидят.

2. Общий режим с backend
------------------------
Запустите start-backend.bat.

После запуска откройте:
http://localhost:3000

В этом режиме все люди сохраняются в общий файл:
data/rodoslovnoe-derevo-data.json

Если другой компьютер откроет этот же backend, он увидит то же самое дерево.
Для компьютеров в одной сети нужно открыть адрес вашего компьютера, например:
http://192.168.1.25:3000

Логин для редактирования:
admin

Пароль:
margoshvili

Публичные посетители могут смотреть дерево без входа. Добавлять, удалять и
сохранять людей можно только после входа.

Как поменять пароль
-------------------
Перед запуском backend можно задать другой пароль через переменные:

set TREE_LOGIN=admin
set TREE_PASSWORD=noviy-parol
node server.js

Важно
-----
GitHub Pages не умеет запускать backend. Если сайт открыт только через GitHub
Pages, он будет публичной страницей без серверного сохранения. Для общего
редактирования нужен запущенный server.js на компьютере или отдельный хостинг
для Node.js.

Файлы
-----
index.html - страница сайта.
server.js - backend и сервер для сайта.
start-backend.bat - запуск backend на Windows.
start-site.bat - старый быстрый запуск index.html.
assets/css/styles.css - внешний вид.
assets/js/app.js - логика дерева.
assets/images/male.png - иконка мужского пола.
assets/images/female.png - иконка женского пола.
data/rodoslovnoe-derevo-data.json - общая база дерева для backend.
