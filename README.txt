Мое родословное дерево
======================

Как открыть:
1. Откройте файл index.html двойным щелчком.
   Можно также открыть start-site.bat.
2. Сайт работает без интернета и без установки программ.
3. Можно скопировать всю папку rodoslovnoe-derevo на флешку.

Важно про сохранение:
- Люди сохраняются в браузере на этом компьютере.
- Положение карточек на дереве тоже сохраняется.
- Чтобы перенести данные на флешку или другой компьютер, нажмите "Сохранить файл".
- Чтобы восстановить данные, нажмите "Загрузить файл" и выберите JSON-файл.

Как двигать дерево:
- Перетаскивайте карточки людей мышью.
- Линии между родителями и детьми перестраиваются автоматически.
- Поколения раскладываются сверху вниз: поколение 1 сверху, поколение 2 ниже.
- Нажмите на карточку человека, чтобы его данные появились во вкладке "Человек".
- Нажмите на пустое место дерева, чтобы очистить вкладку "Человек".
- В форме есть один блок "Тип связи": отец, мать, муж, жена, сын, дочь, дедушка, бабушка.
- В форме есть поле "Пол", чтобы вручную выбрать мужской или женский цвет карточки.
- При клике по человеку слева показываются его имя, годы, заметка и список связей.
- Для связи "Ребёнок" можно выбрать второго родителя, чтобы ребёнок шел от конкретной пары.
- Если у человека указан муж или жена, между ними появляется линия пары.
- При раскладке муж ставится слева, жена справа.
- Если человеку добавить отца или мать, родители уходят на поколение выше, а человек с супругой ниже.
- Если указаны сын или дочь, ребёнок подключается к родителю линией.
- Мужской пол подсвечивается зелёным, женский пол подсвечивается розовым.
- Если двигать одного из родителей пары, второй родитель двигается вместе с ним.
- Дети при этом не двигаются, меняются только линии.
- Кнопка "Разложить" заново расставляет людей по поколениям.
- Кнопки "+" и "-" меняют масштаб.

Файлы сайта:
- index.html - главный файл, который нужно открывать.
- start-site.bat - быстрый запуск сайта на Windows.
- assets/css/styles.css - внешний вид.
- assets/js/app.js - логика добавления, удаления и сохранения.
- data/rodoslovnoe-derevo-data.json - пустой стартовый файл для архива.
